# 🚀 GUIDE DE DÉMARRAGE RAPIDE - NJARIGU

## Bienvenue dans la Révolution Éducative Africaine!

Ce guide va t'aider à démarrer **NJARIGU** même avec zéro compétence en programmation.

---

## 📋 ÉTAPE 1: Installer les Outils Gratuits

### Sur ton Windows (Core i5, 6GB RAM)

#### A. Visual Studio Code (Éditeur de code)
1. Télécharge ici: https://code.visualstudio.com/
2. Installe comme un programme normal (Suivant, Suivant, Terminer)
3. C'est GRATUIT et léger

#### B. Node.js (Pour tester l'application web)
1. Télécharge ici: https://nodejs.org/ (prends la version LTS)
2. Installe avec les options par défaut
3. Redémarre ton ordinateur après installation

#### C. Emscripten SDK (Optionnel - pour compiler C++ vers Web)
*On verra cette étape plus tard quand tu seras à l'aise*

---

## 🎮 ÉTAPE 2: Tester ta Première Démo MAINTENANT

### Option A: Version Web Immédiate (Recommandée)

1. **Ouvre le dossier** `Njarigu_Edu` que j'ai créé pour toi
2. **Double-clique** sur le fichier `web/index.html`
3. **Ça y est!** L'application s'ouvre dans ton navigateur!

🎉 Tu viens de lancer NJARIGU!

Tu peux:
- Cliquer sur "Apprendre" pour voir les leçons
- Cliquer sur "Défis" pour voir les jeux multijoueurs
- Voir ta progression
- Gagner des points et des pièces

### Option B: Version Console (C++ natif)

Si tu veux compiler le code C++:

1. Ouvre Visual Studio Code
2. Fais `Fichier > Ouvrir le dossier` et sélectionne `Njarigu_Edu`
3. Installe l'extension "C/C++" de Microsoft (elle te sera proposée automatiquement)
4. Ouvre le terminal dans VS Code (`Terminal > Nouveau terminal`)
5. Tape ces commandes une par une:

```bash
# Pour compiler avec g++ (si installé)
g++ -std=c++17 -o njarigu.exe src/main.cpp -I include

# Puis lance le programme
njarigu.exe
```

*Note: Si g++ n'est pas installé, on utilisera la version web qui est plus simple.*

## 📚 ÉTAPE 4: Ajouter Tes Cours

### Comment M'envoyer le Contenu Pédagogique

Tu dois me fournir:
1. **Le nom de la leçon** (ex: "Les nombres de 1 à 20")
2. **La matière** (Maths, Français, Culture, etc.)
3. **Le niveau** (Maternelle, CP, CE1, etc.)
4. **Le contenu** (ce que l'enfant doit apprendre)
5. **Les objectifs** (3-5 points clés)
6. **L'idée de jeu** (comment transformer ça en jeu amusant)

### Exemple de Format à Me Donner:

```
Matière: Mathématiques
Niveau: Maternelle (3-5 ans)
Titre: Découvrir les formes géométriques
Contenu: 
  - Le cercle est rond comme une balle
  - Le carré a 4 côtés égaux
  - Le triangle a 3 côtés
  
Objectifs:
  - Reconnaître un cercle
  - Reconnaître un carré
  - Reconnaître un triangle
  
Idée de jeu: 
  "Chasse aux formes" - L'enfant doit trouver dans la classe
  des objets qui correspondent à chaque forme. 
  Chaque bonne réponse donne une étoile.
```

Dès que tu m'envoies ce format, je le code directement dans l'application!
