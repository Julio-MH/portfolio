# Configuration de NJARIGU

## Structure du Projet

```
Njarigu_Edu/
│
├── 📄 README.md              # Présentation du projet
├── 📄 GUIDE_DEMARRAGE.md     # Guide pour commencer (LIS-MOI EN PREMIER!)
├── 📄 package.json           # Configuration du projet
│
├── 📁 src/                   # Code source C++
│   └── main.cpp             # Moteur principal et logique de jeu
│
├── 📁 include/               # Fichiers d'en-tête C++
│   └── engine.h             # Définitions des classes et structures
│
├── 📁 web/                   # Interface web
│   └── index.html           # Application web autonome
│
├── 📁 assets/                # Ressources (à ajouter)
│   ├── images/              # Illustrations, icônes
│   ├── audio/               # Sons, musiques africaines
│   └── fonts/               # Polices d'écriture
│
├── 📁 data/                  # Données et contenus
│   ├── curriculum/          # Cours par matière et niveau
│   ├── users/               # Profils des élèves
│   └── progress/            # Historique de progression
│
└── 📁 build/                 # Fichiers compilés (généré automatiquement)
```

## Technologies Utilisées

### Cœur (C++)
- **Standard:** C++17/20
- **Compilation Web:** Emscripten (WebAssembly)
- **Bibliothèques:** STL standard

### Interface Web
- **HTML5** - Structure
- **CSS3** - Styles et animations
- **JavaScript** - Interactions
- **Canvas API** - Graphismes de jeu

### Base de Données
- **SQLite** - Stockage local
- **JSON** - Échange de données

## Système de Niveaux

### Par Âge/Scolarité
1. **Maternelle** (3-5 ans) - 🌱 Découverte
2. **CP-CE1** (6-7 ans) - 📝 Apprentissage
3. **CE2-CM1** (8-9 ans) - 📚 Consolidation
4. **CM2** (10-11 ans) - 🔍 Approfondissement
5. **6ème-5ème** (11-12 ans) - 🧠 Développement
6. **4ème-3ème** (13-14 ans) - ⚡ Spécialisation
7. **Seconde** (15 ans) - 🎯 Orientation
8. **Première** (16 ans) - 💪 Expertise
9. **Terminale** (17 ans) - 🦅 Maîtrise

### Par Compétence
- 🌱 Débutant
- 🌿 Apprenti
- 🌳 Compétent
- 🦅 Expert
- 🌟 Génie

## Modes de Jeu

### 1. Mode Apprentissage
- Leçons interactives avec graphismes animés
- Explications progressives
- Exercices guidés avec feedback immédiat

### 2. Mode Pratique
- Exercices autonomes
- Répétition espacée pour mémorisation
- Statistiques de progression

### 3. Mode Défi
- **Solo:** Contre l'IA adaptative
- **Duel:** 2 joueurs sur même écran
- **Trio:** 3 joueurs en compétition
- **Équipe:** Coopération pour résoudre des problèmes

### 4. Mode Création
- Projets pratiques
- Application des connaissances
- Partage des créations

## Intelligence Artificielle

### Adaptation Dynamique
Le système analyse en temps réel:
- Taux de réussite
- Temps de réponse
- Types d'erreurs
- Progression dans le temps

### Ajustements Automatiques
- Difficulté des exercices
- Vitesse d'apprentissage
- Types de contenus proposés
- Fréquence des révisions
